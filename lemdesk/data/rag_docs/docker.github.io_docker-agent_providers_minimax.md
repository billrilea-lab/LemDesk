Overview MiniMax provides AI models through an OpenAI-compatible API. docker-agent includes built-in support for MiniMax as an alias provider. Setup Get an API key from MiniMax Set the environment variable: export MINIMAX_API_KEY = your-api-key Usage Inline Syntax The simplest way to use MiniMax: agents : root : model : minimax/MiniMax-M2.5 description : Assistant using MiniMax instruction : You are a helpful assistant. Named Model For more control over parameters: models : minimax_model : provider : minimax model : MiniMax-M2.5 temperature : 0.7 max_tokens : 8192 agents : root : model : minimax_model description : Assistant using MiniMax instruction : You are a helpful assistant.
